Aplicación  de  una calculadora simple en iOS  desarrollado en xcode  con el lenguaje Swift  .
La aplicación permite realizar las 4 operaciones básicas (suma, resta, multiplicación y división),
La aplicación aun cuenta con MUCHOS bugs , como permitir la división con ceros , no valida el  procedimiento.
